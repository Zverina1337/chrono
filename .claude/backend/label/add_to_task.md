# add_label_to_task

Привязка лейбла к задаче. Операция идемпотентна — повторный вызов не вызовет ошибку.

## Tauri-команда

```
add_label_to_task
```

## Входные данные

| Параметр    | Тип      | Описание    |
|-------------|----------|-------------|
| `taskUuid`  | `string` | UUID задачи |
| `labelUuid` | `string` | UUID лейбла |

## Выходные данные

`null` (успех).

## Особенности

- Использует `INSERT OR IGNORE` — дублирование не вызывает ошибку.
- Одна задача может иметь множество лейблов, один лейбл может быть у множества задач.

## Ошибки

| Ошибка               | Когда                      |
|----------------------|----------------------------|
| `Ошибка базы данных` | Проблемы с SQLite-запросом |

## Пример вызова (TypeScript)

```ts
await invoke("add_label_to_task", {
  taskUuid: "...",
  labelUuid: "..."
});
```

## Исходный код

- Команда: `src-tauri/src/commands/label_cmd.rs` → `add_label_to_task`
- Репозиторий: `src-tauri/src/repository/label_repo.rs` → `add_to_task()`
