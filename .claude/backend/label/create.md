# create_label

Создание нового глобального лейбла.

## Tauri-команда

```
create_label
```

## Входные данные

| Параметр | Тип      | Обязательно | Описание                              |
|----------|----------|:-----------:|---------------------------------------|
| `name`   | `string` | да          | Название (должно быть уникальным)     |
| `color`  | `string` | да          | HEX-цвет (напр. `#ef4444`)           |

## Выходные данные

Объект `Label` (см. `get_all.md`).

## Ошибки

| Ошибка               | Когда                                   |
|----------------------|-----------------------------------------|
| `Ошибка базы данных` | Лейбл с таким именем уже существует или другие проблемы с БД |

## Пример вызова (TypeScript)

```ts
const label = await invoke<Label>("create_label", {
  name: "Баг",
  color: "#ef4444"
});
```

## Исходный код

- Команда: `src-tauri/src/commands/label_cmd.rs` → `create_label`
- Репозиторий: `src-tauri/src/repository/label_repo.rs` → `create()`
