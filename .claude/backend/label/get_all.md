# get_all_labels

Получение всех глобальных лейблов (теги глобальные, не привязаны к проекту).

## Tauri-команда

```
get_all_labels
```

## Входные данные

Нет параметров.

## Выходные данные

Массив объектов `Label`:

```json
[
  {
    "uuid": "...",
    "name": "Баг",
    "color": "#ef4444"
  }
]
```

| Поле    | Тип      | Описание                |
|---------|----------|-------------------------|
| `uuid`  | `string` | UUID v4, первичный ключ |
| `name`  | `string` | Название лейбла (уникальное) |
| `color` | `string` | HEX-цвет               |

## Сортировка

По `name ASC`.

## Ошибки

| Ошибка               | Когда                      |
|----------------------|----------------------------|
| `Ошибка базы данных` | Проблемы с SQLite-запросом |

## Пример вызова (TypeScript)

```ts
const labels = await invoke<Label[]>("get_all_labels");
```
