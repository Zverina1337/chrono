# delete_project

Удаление проекта. Каскадно удаляет все связанные статусы, приоритеты, задачи, подзадачи, записи времени и связи с лейблами.

## Tauri-команда

```
delete_project
```

## Входные данные

| Параметр | Тип      | Описание     |
|----------|----------|--------------|
| `uuid`   | `string` | UUID проекта |

## Выходные данные

Ничего (`null`). Успешное выполнение = проект удалён.

## Каскадное удаление

При удалении проекта автоматически удаляются (через `ON DELETE CASCADE`):
- Все статусы проекта (`statuses`)
- Все приоритеты проекта (`priorities`)
- Все задачи проекта (`tasks`)
  - Все подзадачи этих задач (`subtasks`)
  - Все записи времени этих задач (`time_entries`)
  - Все связи задач с лейблами (`task_labels`)

Сами лейблы (`labels`) НЕ удаляются — они глобальные.

## Ошибки

| Ошибка                        | Когда                          |
|-------------------------------|--------------------------------|
| `Не найдено: Проект {uuid}...` | Проект с таким UUID не существует |
| `Ошибка базы данных`           | Проблемы с SQLite-запросом       |

## Пример вызова (TypeScript)

```ts
await invoke("delete_project", { uuid: "550e8400-..." });
```

## Исходный код

- Команда: `src-tauri/src/commands/project_cmd.rs` → `delete_project`
- Репозиторий: `src-tauri/src/repository/project_repo.rs` → `delete()`
