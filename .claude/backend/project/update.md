# update_project

Частичное обновление проекта. Обновляются только переданные поля.

## Tauri-команда

```
update_project
```

## Входные данные

| Параметр | Тип             | Описание           |
|----------|-----------------|--------------------|
| `uuid`   | `string`        | UUID проекта       |
| `data`   | `UpdateProject` | Поля для обновления |

Объект `UpdateProject` (все поля опциональные):

```json
{
  "name": "Новое название",
  "description": "Новое описание",
  "position": 2
}
```

| Поле          | Тип              | Описание                        |
|---------------|------------------|---------------------------------|
| `name`        | `string \| null` | Новое название                  |
| `description` | `string \| null` | Новое описание                  |
| `position`    | `number \| null` | Новая позиция для сортировки    |

## Выходные данные

Объект `Project` — обновлённый проект.

## Логика

1. Проверка существования проекта по UUID
2. Обновление только переданных полей (partial update)
3. `updated_at` обновляется при каждом изменении
4. Возврат обновлённого проекта

## Ошибки

| Ошибка                        | Когда                          |
|-------------------------------|--------------------------------|
| `Не найдено: Проект {uuid}...` | Проект с таким UUID не существует |
| `Ошибка базы данных`           | Проблемы с SQLite-запросом       |

## Пример вызова (TypeScript)

```ts
const updated = await invoke<Project>("update_project", {
  uuid: "550e8400-...",
  data: { name: "Переименованный проект" }
});
```

## Исходный код

- Команда: `src-tauri/src/commands/project_cmd.rs` → `update_project`
- Репозиторий: `src-tauri/src/repository/project_repo.rs` → `update()`
