# delete_status

Удаление статуса. Задачи с этим статусом получат `statusUuid = null` (ON DELETE SET NULL).

## Tauri-команда

```
delete_status
```

## Входные данные

| Параметр | Тип      | Описание     |
|----------|----------|--------------|
| `uuid`   | `string` | UUID статуса |

## Выходные данные

Ничего (`null`).

## Побочные эффекты

Задачи, которые ссылались на этот статус, получат `status_uuid = NULL`. Сами задачи не удаляются.

## Ошибки

| Ошибка                         | Когда                            |
|--------------------------------|----------------------------------|
| `Не найдено: Статус {uuid}...` | Статус с таким UUID не существует |

## Пример вызова (TypeScript)

```ts
await invoke("delete_status", { uuid: "..." });
```

## Исходный код

- Команда: `src-tauri/src/commands/status_cmd.rs` → `delete_status`
- Репозиторий: `src-tauri/src/repository/status_repo.rs` → `delete()`
